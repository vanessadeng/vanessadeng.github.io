Java Basic Introduction#

Welcome to the world of Java, where we aim to teach the most practical technology in the shortest time possible.

Java Background Knowledge

  • Java was developed by Sun Microsystems (Stanford University Network) in 1995. It was initially called Oak before being renamed to Java.
    James Gosling, known as the father of Java, was born on May 19, 1955, in Canada.
  • In 2009, Sun Microsystems was acquired by Oracle Corporation.
  • Java is one of the most popular programming languages worldwide and is extensively used in China for its portability, security, and performance.

Why Java?

  • Rich development community and comprehensive functionalities.
  • Can be used for desktop applications (tax management software, IDEs), enterprise-level applications (microservices, large internet applications), mobile application development (Android, medical devices), server systems, big data development (Hadoop), and game development (Minecraft).

Java Technology System

  • Java SE (Standard Edition): Core and foundation of Java technology.
  • Java EE (Enterprise Edition): A set of solutions for enterprise-level application development.
  • Java ME (Micro Edition): Solutions for mobile device applications.

Quick Start with Java

  • Java Development Environment Preparation: Introduction to JDK, installation, and common commands.
  • First Java Program - HelloWorld: Common issues and execution principle.
  • IDEA Development Tool Usage: Basics of using IntelliJ IDEA, including setting fonts, themes, and shortcuts.

Java Development Steps

  • Writing Code: Create a HelloWorld.java source code file.
  • Compiling Code: Use javac to compile the Java program.
  • Running Code: Execute the compiled class file with java.

Common Java Development Tools

  • Text editors like Notepad++ and IDEs (Integrated Development Environments) like Eclipse, IntelliJ IDEA, and NetBeans enhance development efficiency with features like code intelligence and error prompts.

Introduction to IntelliJ IDEA

  • A preferred tool for many enterprises due to its excellent code error alerts and intelligent code completion features.
  • IDEA simplifies Java development by integrating code writing, compiling, and execution in one place.

Java Programming Basics

  • Variables: Understand the concept, characteristics, and applications.
  • Data Types: Learn about primitive types, operators, and control statements.
  • OOP Concepts: Grasp the basics of Object-Oriented Programming including classes, objects, inheritance, and polymorphism.
